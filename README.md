# Dew on Roses - Django Blog Project

A personal blog platform built with Django, designed to help users create, manage, and share blog posts with ease.

## Features

- User registration and authentication
- Create, update, and delete blog posts
- View posts by different authors
- Responsive design for desktop and mobile
- Environment variable management with `.env`
- Basic line-ending normalization for cross-platform compatibility

## Technologies Used

- Python 3.x
- Django Web Framework
- HTML, CSS, JavaScript
- Git for version control

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Maaz708/Dew_on_Roses_django_blogproject1.git
cd Dew_on_Roses_django_blogproject1

2. create a virtual environment

python -m venv .venv
# On Windows
.\.venv\Scripts\activate
# On macOS/Linux
source .venv/bin/activate

3. Install dependencies

pip install -r requirements.txt

4. Create a .env file based on .env.example and add your environment variables.

5. Run migrations:
python manage.py migrate

6. Start the development server:
python manage.py runserver

7. Open your browser and go to http://127.0.0.1:8000/ to see the blog in action.
