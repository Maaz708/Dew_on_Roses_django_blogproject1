About This Project

This project is a blog platform designed to help users share their thoughts, ideas, and stories with a wider audience. Whether you’re a casual writer or a professional blogger, this platform provides an easy-to-use interface to create, edit, and manage blog posts.

Features

  • User registration and authentication
  • Create, update, and delete blog posts
  • View posts by different authors
  • Responsive design for seamless use on mobile and desktop

Technology Stack

Built with Django for backend, HTML/CSS/Bootstrap for frontend, and SQLite/PostgreSQL as the database.
